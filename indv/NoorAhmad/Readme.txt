A- Tools used in this project:
1- JSP for the server pages.
2- Tomcat server
3- Jena library
4- dbpedia (http://dbpedia.org/sparql) for query

B- How to run the code:
1- Project could be runned by both eclipse and IntelliJ Idea if the Tomcat server and Jena library is installed.
2- You can run index.jsp which is located in WebContent folder.
